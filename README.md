[日本語](./README_ja.md)

A classical music game for GBA.

## Controls

### Title

    START:  Proceed

### Menu

    UP/DOWN:    Move cursor

    A:          Listen

    START:      Play

### Playing

    DirPad-[U/D/L/R]/A/B:   For play

    START:                  Pause/Resume

    Shoulder-[L/R]:         Quit while pause

    SELECT:                 See menu after end

## Requirement

GBA SP (with Optimize Boot cable USB), mGBA 0.9.2
